## Automated Testing Features
This project implements **Integration Testing** to ensure API reliability. The tests cover:
1.  **Server Status:** Verifies the application handles HTTP requests correctly (200 OK).
2.  **API Validation:** Ensures the `/api/projects` endpoint returns a valid JSON array with project data.
